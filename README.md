# Phishing Email Analysis Practice

## Description
This project focuses on analyzing real-world phishing email samples to identify
malicious indicators such as forged headers, failed SPF/DKIM checks, malicious URLs,
and redirection chains. The goal is to understand common phishing techniques and
improve email threat detection skills.

## Objectives
- Analyze email headers
- Verify SPF, DKIM, and DMARC results
- Identify malicious URLs and redirections
- Document Indicators of Compromise (IOCs)
- Understand phishing tactics and patterns

## Skills Demonstrated
- Email header analysis
- Threat intelligence
- Cybersecurity analysis
- SOC-level investigation

## Tools Used
- Email header analyzers
- WHOIS tools
- URL inspection tools
- Threat intelligence platforms

## Use Case
SOC analyst training, cybersecurity labs, and academic security projects.
